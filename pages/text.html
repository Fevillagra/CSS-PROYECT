Propiedad font-family

El valor asignado a la propiedad font-family esta compuesto por dos propiedades especificas.
Font Family:El nombre de la familia de la fuente. Por ejemplo, 'Times' y 'Helvética' son nombres de familia.Los nombres de familia que contengan espacios, deben escribirse entre comillas (por ejemplo: "Times New Roman").
Generic Family:Se han definido los siguientes nombres genéricos: serif, sans-serif, cursive, fantasy, monospace. Los nombres genéricos son palabras claves y no deben ponerse entre comillas.

Ej:
.p {
  font-family: "Gill Sans Extrabold", Helvetica, sans-serif;
}

Hola Mundo!

Aleternativas

La propiedad font-family puede alojar como valor más de un solo tipo de fuente especificado que serivirá en caso de que el navegador no contenga la fuente principal indicada. El navegador, entonces, tomará la segunda fuente como valor para asignar al elemento.

Ej:
.p {
  font-family: Arial, "Helvetica Neue", Helvetica, sans-serif;
}

Hola Mundo!


Si el navegador no admite la fuente Arial, prueba con las siguientes fuentes (Helvetica Neue, luego Helvetica). Si el navegador no tiene ninguno de ellos, probará el genérico sans-serif.

Propiedad font-size

La propiedad font-size establece el tamaño de la letra. Estas se pueden establecer mediante el uso de palabras clave como por ejemplo: xx-small, small, medium, large larger, xlarge, etc.
Támbien se pueden usar valores númericos en pixeles, rems o ems. "em" al ser un tipo de medida relativa(al igual que rem) se dimencionara acorde al tamaño de otro elemento cuyo tamaño haya sido definido previamente. Mientras que em es la medida tomada del elemento pádre más lejano, rem es la medida subjetiva tomada del elemento padre contenedor. Si no se ha establecido el tamaño de fuente en ninguna parte de la pagina se utilizara el tamaño por defecto de 16 px. Por lo tanto em = pixels/16

Valores: rem, em, px, palabras clave.

Ej:
.h1 {
  font-size: 3rem;
}

Hola Mundo!

Propiedad font-style

La propiedad font-style es usada típicamente para especificar el texto en cursiva. Teniendo solo 3 valores: normal, italic y oblique

Ej:
.p {
  font-style: oblique;
}

Hola Mundo!

Propiedad font-weight

La propiedad font-wight controla el grosor o espesor del texto. Los valores pueden ser establecidos como: lighter, normal, bold, bolder.
Tambien se puede definir el grosor de la ffuente mediante el conjunto de números que va desde 100(delgado) hasta 900(grueso) siendo 400 normal y 700 bold como valores de referencia.

Ej:
.p {
  font-weight: 700;
}

Hola Mundo!

Propiedad font-variant

La propiedad font-variant permite usar en todo el texto "small caps"(mayúsculas pequeñas), los valores que se pueden usar son normal, small-caps e inherit(heredado)

Ej:
.p {
  font-variant: small-caps;
}

Hola Mundo!

line-height

La propiedad CSS line-height establece la altura de una casilla remarcada por líneas. Comúnmente se usa para establecer la distancia entre líneas de texto.
Si el valor es numerico, será dividido en dos ya que el line-height nace en el medio. El valor de 1 sera igual al tamaño de la letra.

Propiedad color

La propiedad color especifica el color de texto siendo posible elegir su valor mediante su nombre en inglés tales como, rojo(red), azul(blue), etc; o a través de valores hexadecimales y RGB
Hexadecimal: Comienza con un signo # seguido de 6 valores hexadecimales que van desde 0 a FF correspondiendo los dos primeros valores a rojo, los terceros y cuartos a verde y los restantes azul. Ej: #FF0000 (rojo) También se puede expresar de la siguiente manera: #fff #fffo #ffffffoo Donde o representa la opacidad.
RGB: De las siglas(red,green,blue), define valores individuales para cada uno de los colores. Variando de 0 a 255. Ej: rgb(255, 0,0) (rojo)

Ej:
.p {
  color: rgb(255, 0,0)
}

Hola Mundo!

Propiedad text-align

La propiedad text-align especifica la alineación horizontal del texto en un elemento; el texto en una página web es alineado hacia la izquierda. Existen tres tipos de valores acordes a la alineación del texto: left, center y right

Ej:
.p {
  text-align: center;
}

Hola Mundo!

Propiedad vertical-align

La propiedad vertical-align establece la alineación vertical de un elemento, los valores comunmente usados son: top, middle y bottom

Ej:
.top {
  vertical-align: top;
}


.middle {
  vertical-align: middle;
}


.bottom {
  vertical-align: bottom;
}


top middle bottom

Otros valores

La propiedad vertical-align tambien acepta otros valores, aunque menos usados: baseline, sub, super, px, pt(puntos) y %
baseline: Valor por defecto.
Sub: Añade un subtexto.
super: Añade un supertexto.

Ej:
span {
  vertical-align:-10px;
}

Hola mundo!

Alineamiento vertical en divs

Para que el alineamiento vertical funcione tambíen en otros elementos es necesario aplicar un valor a la propiedad display del elemento padre y otro al hijo.
En el div padre: display:inline-table
En el div hijo: display: table-cell + la propiedad con la ubicacion

Ej:
div.padre {
  display: inline-table;
  height: 100px;
}


div.hijo {
  display: table-cell;
  vertical-align: middle;
}

Propiedad text-decoration

Como su nombre sugiere, la propiedad text-decoration especifica como será decorado el texto.
Los valores son: none(por defecto), inherit(heredado del padre), overline(linea sobre el texto), underline(linea bajo el texto), line-through(linea atravesada).

Ej:
.p{
  text-decoration: underline;
}


Hola Mundo!

Propiedad text-indent

La propiedad text-indent especifica cuanto espacio debe ser dejado antes de empezar la primer línea de texto(sangría). Los valores de la propiedad son los propios de length(longitud) [px, pt, cm, em,] % e inherit(heredado).
Si el valor es negativo el texto será indentado hacía la izquierda.

Ej:
.p{
  text-indent: 10px;
}


Hola Mundo!

Propiedad text-shadow

La propiedad text-shadow añade sombreado al texto. Puede recibir 4 valores: el primero indica el posicionamiento horizontal de la sombra, el segundo el vertical, el tercero; la cantidad de desenfoque de la sombra; y el cuarto el color de la misma.
Para agregar más de una sombra al elemento se separan el conjunto de valores dentro de la propiedad por una coma. Los valores negativos permitiran invertir la dirección de la sombra. Las sombras aparecen por orden de establecimientos es decir, si primero se establece la roja y luego la azul, la primera estará superpuesta a la segunda.

Ej:
.p{
  text-shadow: 10px 10px 0 red, 10px 10px 1px blue;
}


Hola Mundo!

Propiedad text-transform

La propiedad text-transform especifica el estilo del texto en cuanto a minúsculas y mayúsculas.
Los valores son 3: capitalize(las iniciales se muestran como mayúsculas), uppercase(todo en mayúsculas) y lowercase(todo en minúsculas).

Ej:
.p{
  text-transform: uppercase;
}


Hola Mundo!

Propiedad letter-spacing

La propiedad letter-spacing determina el espacio entre caracteres en un texto.


Valores: normal(viene por defecto), length(se especifican medidas de espacio como px, pt, em, etc.) e inherit(heredado).

Los valores length pueden ser tanto positivos y negativos, mientras más grande sea el número negativo, menos espacio habrá entre caracteres.

Ej:
.p{
  letter-spacing: -5px;
}


Hola Mundo!

Propiedad word-spacing

La propiedad word-spacing especifica el espacio entre palabras al igual que letter word-spacing los valores que puede recibir son normal, length & inherit.

Ej:
.p{
  word-spacing: 20px;
}


Hola Mundo!

Propiedad white-space

La propiedad white-space especifica como el espacio en blanco será manejado dentro del texto.
Los valores que puede tomar son: normal(por defecto), inherit(heredado), nowrap( hace que el texto transcurra en una misma línea hasta que se encuentre con una etiqueta br), pre(los espacios en blanco acumulados dentro de la etiqueta y los saltos de línea serán tomados en cuenta), pre-line(dentro de la etiqueta,la acumulación de espacios en blanco es ignorada pero los saltos de línea son tomados en cuenta) y pre-wrap( El valor pre-wrap se comporta como el valor pre, excepto que agrega saltos de línea adicionales para evitar que el texto se salga del cuadro del elemento.)

Ej:
.p{
  white-space: pre;
}


Hola Mundo!

Propiedad word-wrap

La propiedad word-wrap permite que las palabras largas se dividan y continuen en la siguiente línea toma dos valores normal y break-word siendo este último por defecto. Si el valor es normal y la palabra es demasiado larga, esta continuara en la misma línea transgrediendo el limite del contendor.

.cont {
  width: 210px;
  height: 100px;
  border: 1px solid #000;
  word-wrap: normal;
}

La regla @font-face

@font-face permite cargar fuentes a la página web diferentes a las que vienen por defecto. La url debe apuntar a un archivo OpenType integrado (eot), mientras que Firefox, Chrome, etc. admiten fuentes True Type (ttf) y fuentes OpenType (otf)

Ej:
@font-face {    font-family: Delicious;
   src: url('Delicious-Roman.otf');
}
@font-face {
  font-family: Delicious;
  font-weight: bold;
  src: url('Delicious-Bold.otf');
}

Hola Mundo!

Propiedad direction

La propiedad direction se utiliza para indicar en que dirección fluye el texto: rtl para hebreo o árabe y ltr para otros tipos de escritura. Esto tendría que especificarse como parte del documento (por ejemplo, usando el código dir en HTML) en lugar que directamente en la hoja de estilo CSS.
